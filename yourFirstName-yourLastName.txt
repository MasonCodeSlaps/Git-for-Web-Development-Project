    7. What is an id? 
        An id is a CSS attribute similar to the class, difference is the ID should be used for stuff that you want solo styled.

    8. What elements can I add an id to? 
        Theoretically anything you want, should be used for items you want to style alone.

        Class is basically a simpler version of ID, used almost the same way except more used overall. 

    11. What elements can I add a class to?

    12. How many times can I use the same class on a page?
